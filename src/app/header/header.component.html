SciFind
What is SciFind?
SciFind is a light weight search engine for scientific research and a pilot project of PunktStrich. It does not track your behaviour nor your inputs, it simply makes an http-request to the data provider and prepares the data to be displayed in a visual appealing, clear way.

How does it work?
First a search query needs to be entered into the input. By pressing one of the buttons the user decides which database should be searched. Momentarily, SciFind provides three different data sources:
  • IEEE
  • Elsevier
  • Springer
After the entries have been loaded, the user can scroll for sources of interest and by clicking on the title be directed to the original website to access the full article.

Future plans?
As this side is a pilot project, more features might be added later if there is enough resonance. Planned features do include supplying a bibtex format for quickly adding the sources to the user's personal reference management library and further data sources. However, the latter might be problematic since not all research platforms do provide open api access. For questions, feedback or complaints, do not hesitate to contact us.
